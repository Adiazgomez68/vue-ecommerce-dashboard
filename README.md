# E-commerce Dashboard - Vue.js

Product Dashboard for e-commerce built with Vue 3, TypeScript and Pinia.

## Technologies

- Vue 3 (Composition API)
- TypeScript
- Pinia (State Management)
- Vue Router
- FakeStore API

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Features

- ✅ Product list
- ✅ Search by name
- ✅ Filter by category
- ✅ Pagination
- ✅ Detail view

## Structure

- `/components` - Reusable components
- `/views` - Main views
- `/stores` - State management with Pinia
- `/services` - API services
- `/types` - TypeScript definitions
